int XOcp_ExtendPcr ( XOcp_RomHwPcr  PcrNum,
u64  ExtHashAddr 
)

This function extends the PCR with provided hash by requesting ROM service.

Parameters
PcrNumis the variable of enum XOcp_RomHwPcr to select the PCR to be extended.
ExtHashAddris the address of the buffer which holds the hash to extended.
Returns
  • XST_SUCCESS - If PCR extend is success
  • XST_FAILURE - Upon any failure
